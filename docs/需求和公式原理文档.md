# 需求和公式原理文档

本文档只保留各功能的用途说明与公式原理，用于后续重构时作为需求与计算依据。控件布局、显示规则、页面结构与验收口径由单独的 UI 文档维护。

## 1. 功能用途

### 1.1 ND曝光计算器

- 用途：根据 ND 档位、基础快门和曝光补偿，计算装滤镜后的最终快门时间。

### 1.2 大光比助手

- 用途：用于借助遮挡物单独控制画面高光部分曝光时间的辅助计算，当前开发中。

### 1.3 景深计算器

- 用途：根据画幅、焦距、光圈和对焦距离，计算景深近界、远界、全景深和超焦距。

### 1.4 放大倍率换算

- 用途：用于微距或近摄场景下的放大倍率换算，当前开发中。

### 1.5 色温与色相预览

- 用途：根据色温计算预览色彩，帮助判断不同色温下的视觉效果。

### 1.6 参数斗蛐蛐助手

- 用途：用于不同镜头在常用焦段、不同光圈下的分辨率、二手价格、色散控制、放大倍率、炫光控制等参数对比，帮助用户在有限预算下完成购镜选择。
- 说明：该模块优先提供镜头横向对比与预算建议，不负责具体成像计算的最终判定，结果用于辅助决策。

### 1.7 摄影小知识

- 用途：用于展示摄影相关知识、提示或图文说明，当前待开放。

## 2. 公式原理

### 2.1 曝光计算

基础 EV100：

$$EV = \log_2\left(\frac{N^2}{t}\right)$$

EVISO：

$$EV_{ISO} = \log_2\left(\frac{100 \cdot N^2}{t \cdot S}\right)$$

反推快门：

$$t = \frac{100 \cdot N^2}{2^{EV_{ISO}} \cdot S}$$

其中：

- $N$：光圈系数
- $t$：快门时间（秒）
- $S$：ISO

#### ND 映射

ND 采用硬编码映射优先，未命中时回退到对数计算。

| ND 标签 | 档数 |
| --- | --- |
| ND2 | 1 |
| ND4 | 2 |
| ND8 | 3 |
| ND16 | 4 |
| ND32 | 5 |
| ND64 | 6 |
| ND512 | 9 |
| ND1000 | 10 |
| ND32000 | 15 |

自定义 ND 因子兜底公式：

$$\Delta EV = \log_2(ND\ Factor)$$

### 2.2 景深计算

超焦距：

$$H = \frac{f^2}{N \cdot c} + f$$

景深近界：

$$D_n = \frac{s \cdot (H - f)}{H + s - 2f}$$

景深远界：

$$D_f = \frac{s \cdot (H - f)}{H - s}$$

全景深：

$$\Delta D = D_f - D_n$$

其中：

- $f$：物理焦距
- $N$：光圈系数
- $s$：对焦距离
- $c$：弥散圆直径

### 2.3 色温预览

色温到 RGB 采用 Tanner Helland 拟合公式，结果在 `[0,255]` 内 clamp 并取整。
