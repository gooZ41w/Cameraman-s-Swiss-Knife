# 需求和公式原理文档

本文档作为产品需求、公式定义、显示口径与验收依据的统一来源。当前页面以曝光、景深和色温预览为主；未实现但仍保留的需求集中在文末“历史保留项”中。

## 1. 现有范围

- ND曝光计算器
  - 摇拍助手（开发中）
- 景深计算器
  - 放大倍率换算（开发中）
- 色温预览
- 首页仅负责入口导航

## 2. 通用规则

- 核心公式只允许放在 `src/domain/`。
- 页面层只负责输入、显示、单位换算和格式化。
- 结果展示优先显示计算值，再显示容差或辅助说明。
- 结果进入无穷远或极端值时，统一用 `∞` 作为友好显示。
- 阈值、公式、显示格式和容差口径一旦变化，必须同步更新文档、代码和测试。

## 3. 曝光计算

### 3.1 公式

基础 EV100：

$$EV = \log_2\left(\frac{N^2}{t}\right)$$

EVISO：

$$EV_{ISO} = \log_2\left(\frac{100 \cdot N^2}{t \cdot S}\right)$$

反推快门：

$$t = \frac{100 \cdot N^2}{2^{EV_{ISO}} \cdot S}$$

其中：

- $N$：光圈系数
- $t$：快门时间（秒）
- $S$：ISO

### 3.2 ND 映射

ND 采用硬编码映射优先，未命中时回退到对数计算。

| ND 标签 | 档数 |
| --- | --- |
| ND2 | 1 |
| ND4 | 2 |
| ND8 | 3 |
| ND16 | 4 |
| ND32 | 5 |
| ND64 | 6 |
| ND512 | 9 |
| ND1000 | 10 |
| ND32000 | 15 |

自定义 ND 因子兜底公式：

$$\Delta EV = \log_2(ND\ Factor)$$

### 3.3 当前实现约束

- 当前曝光页固定以“快门”为求解目标。
- 页面不提供设置页入口。
- ND 调整后的快门时间达到 4 秒时，逻辑上进入长曝光区间。
- 快门显示优先使用常见分母的精确标签；大于 60 秒时使用 `hh:mm:ss`。

### 3.4 曝光结果显示规则

- 大字显示计算值。
- 下方小字显示容差信息。
- 容差较大值使用绿色。
- 容差较小值使用红色。

## 4. 景深计算

### 4.1 单位规则

- domain 层统一使用毫米。
- UI 层可以用米展示，但进入计算前必须换算为毫米。
- 对焦距离拖到 400m 时，UI 显示为 `∞`，但参与计算的值仍是 `400000mm`。

### 4.2 公式

超焦距：

$$H = \frac{f^2}{N \cdot c} + f$$

景深近界：

$$D_n = \frac{s \cdot (H - f)}{H + s - 2f}$$

景深远界：

$$D_f = \frac{s \cdot (H - f)}{H - s}$$

全景深：

$$\Delta D = D_f - D_n$$

其中：

- $f$：物理焦距
- $N$：光圈系数
- $s$：对焦距离
- $c$：弥散圆直径

### 4.3 当前输入边界

- 焦距：4mm 到 800mm
- 光圈：f/0.95 到 f/32
- 对焦距离：0.1m 到 400m

### 4.4 容差规则

- 使用 `±5%` 几何容差。
- 大字显示计算值。
- 小字显示容差较大值和较小值。
- 较大值为绿色，较小值为红色。

## 5. 色温预览

### 5.1 色温到 RGB

色温预览采用 Tanner Helland 拟合公式，结果在 `[0,255]` 内 clamp 并取整。

### 5.2 当前范围

- 2000K 到 10000K
