# 用户界面操作文档

本文档描述当前应用的真实页面、入口和交互方式，供使用者与验收人员参考。

## 1. 首页

首页标题为“摄影小助手”。当前可进入的功能有：

- 曝光计算器
- 景深计算器
- 色温与色相预览

页面下方保留“斗转辅助（待开放）”和“摄影小知识（待开放）”两条占位信息。

## 2. 色温与色相预览

### 2.1 操作方式

- 使用色温滑块调整 Kelvin 值，范围为 2000K 到 10000K。
- 页面会实时更新预览色块。
- 点击预览色块进入全屏显示。
- 再次点击全屏覆盖层退出。

### 2.2 使用提示

- 色温数值越低，颜色越偏暖。
- 色温数值越高，颜色越偏冷。

## 3. 曝光计算器

### 3.1 页面结构

页面从上到下分为：

- ND 滤镜档位
- 基础快门
- 曝光补偿
- 结果区

当前页面没有设置页入口。

### 3.2 操作方式

- 使用左右按钮切换 ND 滤镜档位。
- 使用左右按钮切换基础快门。
- 使用左右按钮切换曝光补偿。
- 页面会实时计算装入滤镜后的快门时间。

### 3.3 显示规则

- `装入滤镜后的快门时间` 位于白色结果框中，标题和数值都居中显示。
- 结果数字采用金色。
- 如果结果超过 60 秒，会显示为 `hh:mm:ss`。
- 如果结果进入长曝光区间，页面逻辑会按长曝光处理。
- `当前 EV100 / EVISO` 卡片显示基础曝光值。

### 3.4 ND 显示

ND 选项只显示简洁标签，例如：

- ND2
- ND4
- ND32
- ND1000

档位说明会显示在选项值下方，并使用更轻的灰字。

## 4. 景深计算器

### 4.1 页面结构

景深页从上到下分为：

- 画幅选择
- 焦距
- 光圈
- 对焦距离
- 结果区

### 4.2 操作方式

- 画幅使用下拉框选择。
- 焦距滑块范围为 4mm 到 800mm。
- 光圈滑块范围为 f/0.95 到 f/32。
- 对焦距离滑块范围为 0.1m 到 400m。

### 4.3 结果区说明

结果区分为三段：

- 上方并排显示景深近界和景深远界
- 中间居中显示全景深
- 下方居中显示超焦距

### 4.4 特殊显示规则

- 对焦距离拖到 400m 时，输入显示为 `∞`，但内部计算仍按 400000mm 处理。
- 结果旁的容差说明采用两层结构：大字显示计算值，小字显示容差上下界。
- 容差较大值显示为绿色，较小值显示为红色。

## 5. 通用操作提示

- 所有返回按钮都会把页面带回首页。
- 所有滑块改动都会立即刷新结果。
- 如果看到 `∞`，表示该项已经进入无穷远显示规则，不代表内部计算失效。
