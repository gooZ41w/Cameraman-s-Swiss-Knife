# 页面内容与控件规范

本文档只记录页面层级、功能用途、输入控件和输出控件，不包含视觉设计、显示规则、交互细则或实现约束。它用于后续逐页重构时做内容清单对照。

## 1. 页面层级

### 1.1 主页面

- 作用：统一入口导航。
- 内容：应用标题、用户自定义相机机型展示区、功能入口列表、相机图标自定义入口。
- 说明：主页面只负责跳转，不承载业务计算。

### 1.2 功能页面

- 作用：承载单一功能的输入与输出。
- 内容：返回主页图标、功能标题、输入控件区、输出控件区、简短说明。
- 说明：每个功能页面彼此独立，不作为其它页面的子项。

### 1.3 主页面相机图标自定义页面

- 作用：管理主页面展示的相机图标或相机机型图。
- 内容：返回主页图标、页面标题、导入图标、当前图标预览、预设图标列表。
- 说明：该页面只负责主页面相机图标选择，不承载摄影业务计算。

#### 交互方式

- 三个输入控件都采用左右拖动滚动的方式完成档位变化。
- 左右相邻参数只作为预览，不承担点击切换功能。
- 视觉上保持三档位窗口：上一档、当前档、下一档。
- 交互底层采用虚拟循环滚轮，所有预设档位按循环序列参与拖动。
- 拖动结束后根据拖动距离和速度计算跨越档数。
- 单次拖动最大跨越 `2` 个档位。
- 滚动序列必须是真循环，不允许在循环边界出现视觉回跳。
- 选中的当前档位使用主题色标记。
- 当前档位与相邻档位需要保持足够间距，避免长档位名称拥挤。
- ND 档位、基础快门、曝光补偿的切换方式保持一致，减少学习成本。
- 输入控件只负责切换参数，不在控件内部展示计算结果。
- 每个输入控件展示上一档、当前档、下一档三列内容。

#### 输出控件

- ND 后快门结果
- B门或T门提示

#### 输出交互

- 输出区位于输入参数内容下方。
- 输出区主内容为装入滤镜后的快门时间。
- 当结果小于等于 60 秒且不使用 `hh:mm:ss` 时，显示为最接近的标准基础快门档位。
- 当结果超过 60 秒时，自动切换为 `hh:mm:ss` 格式。
- 当结果使用 `hh:mm:ss` 格式时，下方显示提示：请使用相机的B门或T门模式。
- 输出格式只改变显示方式，不改变计算结果本身。

### 3.3 景深计算器

#### 页面结构

- 顶部工具栏
- 输入参数区
- 输出结果区

#### 顶部工具栏

- 左侧：返回主页图标 `<`
- 中间：页面标题 `景深计算器`
- 右侧：空位占位，保持标题居中

#### 输入控件

- 画幅选择
- 焦距输入
- 光圈输入
- 对焦距离输入

#### 输入控件参数

- 画幅选择：Full Frame、APS-C (Sony/Fuji/Nikon)、APS-C (Canon)、m4/3、1 Inch、自定义画幅
- 自定义画幅：点击后弹出悬浮窗口，显示“请输入画幅的长度和宽度（mm）”，允许输入长度和宽度，点击取消或确认返回。
- 焦距输入：范围 `1mm` 到 `6000mm`，步进 `1mm`
- 光圈输入：范围 `f/0.95` 到 `f/32`，步进 `0.05`
- 对焦距离输入：范围 `0.01m` 到 `400m`，步进 `0.01m`
- 当对焦距离为 `400m` 时，页面显示为 `∞`

#### 数值自定义输入

- `焦距（mm）:` 后方显示值可点击，点击后弹出自定义数值悬浮窗。
- `光圈（f-number）:` 后方显示值可点击，点击后弹出自定义数值悬浮窗。
- `对焦距离（米）:` 后方显示值可点击，点击后弹出自定义数值悬浮窗。
- 可点击数值背后使用轻微加深的底框提示用户可以点击。
- 弹窗输入确认后，对应滑条同步位移到该数值。
- 焦距输入小于 `1mm` 时回弹到 `1mm`，大于 `6000mm` 时回弹到 `6000mm`。
- 光圈输入小于 `f/0.95` 时回弹到 `f/0.95`，大于 `f/32` 时回弹到 `f/32`。
- 对焦距离输入小于 `0.01m` 时回弹到 `0.01m`，大于 `400m` 时回弹到 `400m`，并显示为 `∞`。

#### 输出控件

- 清晰范围摘要
- 展开计算值按钮
- 折叠计算值：景深近界、景深远界、全景深、超焦距
- 超焦距说明：当对焦距离大于超焦距时，在超焦距卡片中显示 `对焦距离大于超焦距，焦点后的景深为∞`

#### 输出交互

- 输出区位于输入参数内容下方。
- 默认只显示清晰范围摘要。
- 当全景深不为无限大时，摘要分三行居中显示：
  - `主体前方xxm`
  - `至主体后方xxm为`
  - `清晰范围`
- 当全景深为无穷大时，摘要分两行居中显示：
  - `距离您前方xxm开外皆为清晰范围`
  - `距离主体前xxm为清晰范围`
- 摘要文案不显示逗号、句号等标点符号。
- `您` 和 `主体` 使用强调色标记。
- 距离数值 `xxm` 使用主题色标记。
- 下方提供 `展开计算值` 选项。
- 点击展开后显示景深近界、景深远界、全景深、超焦距。
- 景深远界或全景深为无穷远时显示为 `∞`。
- 有限距离小于 `1000mm` 时以 `mm` 显示。
- 有限距离大于等于 `1000mm` 时以 `m` 显示。

## 4. 重构时的使用方式

- 先按页面层级创建壳组件。
- 每个页面只补齐本节对应的输入控件和输出控件。
- 页面内容未确定时，先保留空白占位区，再逐步补业务逻辑。
